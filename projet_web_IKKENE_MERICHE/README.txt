# Escape-game

Ce projet est notre escape game géographique nommé GéoAttack sur le thème de Mars Attack film iconique.

Pour installer notre projet il vous suffit de :

installation:

Télécharger le code au format ZIP,
Dézipper le dossier téléchargé,
Lancer MAMP puis accéder à PHPmyadmin
Creer une nouvelle database dans PHPmyadmin. Nommez la "geoattack", puis importer la BDD du jeu (fichier .sql)
Le file directory de MAMP doit se terminer par geoattack_escape_game/code

Pour jouer rentrez l'url "http://localhost/index.php"

La page de login s'affiche alors si c'est votre 1ère connexion cliquez sur enregistrer et mentionnez votre nom d'utilisateur et votre mot de passe.
Vous arriverez alors sur la page d'accueil.
Le but du jeu et le lore sont expliquées dans "But du jeu et lore".
Vous pouvez jouer dans "jouer".
Vous pouvez consulter le tableau des scores dans "Hall of fame".

Bonne destruction !
